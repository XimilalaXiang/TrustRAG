# 引用与审核

引用系统是 TrustRAG 区别于普通 RAG 工具的核心特性。

## 引用系统

### 引用标记

每段 AI 回答中都会包含引用标记，格式如 `[1]`、`[2]`。每个标记对应一段原文内容。

### 引用详情

点击引用标记可以查看：
- 来源文档名称
- 具体页码和段落位置
- 原文内容摘录
- 相关度评分

### 引用跳转

点击"查看原文"可直接跳转到文档的对应位置，方便上下文验证。

## 审核工作流

TrustRAG 提供内置的文档审核功能，支持对 AI 回答的逐段校对。

### 审核状态

| 状态 | 说明 |
|------|------|
| Pending | 待审核 |
| Approved | 已确认准确 |
| Rejected | 标记为不准确 |
| Needs Review | 需要进一步核查 |

### 审核流程

1. 查看 AI 回答
2. 逐条检查引用来源
3. 对每段引用标记审核状态
4. 不准确的引用可以添加批注

## 为什么需要引用？

- **可信度** — 用户可以验证 AI 回答是否有依据
- **透明度** — 清楚知道答案来源于哪些文档
- **可追溯** — 出现问题时可以追溯到原始材料
- **合规性** — 满足对信息来源可追溯的合规要求
