# TrustRAG vs Open WebUI 对比（2026-05-14）

## 结论

TrustRAG 和 Open WebUI 定位有重叠但差异明显。

## 核心差异

| 维度 | Open WebUI | TrustRAG |
|------|-----------|----------|
| 定位 | LLM 聊天界面 | 可信知识工作台 |
| 核心交互 | 对话 | 文档 + 对话 + 复核 |
| RAG | 附加功能 | 核心功能 |
| 引用溯源 | 简单标记 | chunk_id 级精确引用 |
| 原文跳转 | 无/简单 | PDF 页码高亮、段落定位 |
| 人工复核 | 无 | 7 种复核状态 + 审计 |
| 文档管理 | 简单上传 | 完整资料库 |

## 差异化必须落地的三个功能

1. 引用精确度
2. 原文跳转
3. 复核流程

## 关于幻觉问题

- TrustRAG **不能**"系统解决 AI 幻觉问题"——这是理论限制
- 能做到的是"让幻觉可见、可管理、可审计"
- 本质是"幻觉管理系统"，不是"幻觉消除系统"
- 产品定位应诚实表达，不过度承诺
