# TrustRAG UI 设计风格（2026-05-14）

> 状态：已确认
> 参考标杆：Perplexity AI

## 设计理念

**学术 + 可信 + 现代**
- 比 ChatGPT 更严肃，比 Google Scholar 更现代
- 减少装饰，内容为王
- 每一个引用都可追溯、可验证

## 核心 UI 组件设计

### 1. 引用系统（核心差异化）
```
AI 回答文本 [1] 继续文本 [2] 更多内容 [3]。

┌─────────────────────────────────────────┐
│ [1] 📄 技术规范.pdf · 第12页 · 第3章    │
│     "原文摘要片段..."                    │
│                             相关度: 92%  │
├─────────────────────────────────────────┤
│ [2] 📄 设计文档.docx · 第5页            │
│     "原文摘要片段..."                    │
│                             相关度: 87%  │
└─────────────────────────────────────────┘
```

- 引用编号：上标，teal/blue 强调色
- Hover 引用编号 → 弹出来源预览卡
- 点击引用 → 跳转到 PDF 对应位置（高亮）
- 来源卡片：文档名 + 页码 + 章节 + 相关度 + 摘要

### 2. 聊天区域
- 单栏居中布局，最大宽度 720px
- 消息气泡：AI 左对齐（无背景），用户右对齐（浅色背景）
- 流式输出：逐字渲染 + 引用标记实时出现
- 打字指示器：三点动画

### 3. 侧边栏
- 可折叠（宽屏展开，窄屏图标模式）
- 工作区列表
- 文档库快速访问
- 对话历史

### 4. 文档查看器
- PDF 阅读器（Syncfusion）
- 高亮引用片段（黄色半透明覆盖）
- 页码导航
- 搜索功能

### 5. 跟进问题
- 答案下方 pill 按钮
- AI 建议的相关问题
- 点击直接发送

### 6. 审核系统（Phase 2）
- 引用旁的审核徽章（✓ 已验证 / ⚠ 待审核 / ✗ 有异议）
- 审核面板：逐条引用审核

## 动效规范

|------|------|------|
| 按钮状态切换 | 150ms | ease-out |
| 卡片 hover | 200ms | ease |
| 侧边栏展开/折叠 | 250ms | ease-in-out |
| 引用预览弹出 | 200ms | ease-out |
| 流式文字渲染 | 15-30ms/字符 | linear |
| 页面路由切换 | 300ms | ease |

## 无阴影原则

- 不使用 box-shadow
- 通过色彩层级（白→灰→深灰）和细边框创造层级感
- 扁平 + 层叠 = 清晰但不沉重

## 响应式断点

| 名称 | 宽度 | 布局 |
|------|------|------|
| 桌面宽屏 | ≥1200px | 侧边栏 + 内容 + 来源面板 |
| 桌面标准 | 900-1199px | 侧边栏 + 内容 |
| 平板 | 600-899px | 折叠侧边栏 + 内容 |
| 手机 | <600px | 底部导航 + 全屏内容 |
